# VA2022a 编译项目

本项目用于编译 Virtual Acoustics (VA) v2022a，包含完整的版本管理和编译文档。

## 项目结构

```
VA-build/
├── source/          # VA2022a 源代码（Git子模块）
├── dependencies/    # 外部依赖项
├── build/          # CMake 构建目录
├── dist/           # 最终输出/发行版
└── docs/           # 编译文档
```

## 快速开始

详细的编译指南请参见 `COMPILATION_GUIDE.md`

## 版本信息

- VA版本: v2022a
- 构建系统: CMake + Visual Studio 2022
- Matlab版本: R2024a
